Provides documentation on the GERTY setup and usage for users.

GERTY is a robotic OLP (Offline Programming) software framework built on the Rhino3D Grasshopper environment. With GERTY, users can perform custom robotic fabrication using industrial robots for processes such as 3D Printing, WAAM (Wire Arc Additive Manufacturing), and Milling. It enables toolpath control, simulation, and offline programming for robotic machining.
